---

## 1. 概要

### 1.1 目的

本文書は、旧 IAMS（IT Asset Management System）から AEGIS-SIGHT への機能移植状況を管理するチェックリストである。移植対象の5機能と移植しない（廃棄対象の）4機能を明確に区別し、各機能の移植完了状態を追跡する。

### 1.2 移植方針

| 方針 | 説明 |
|------|------|
| 選択的移植 | 必要な機能のみを AEGIS-SIGHT に移植 |
| 再設計 | 単純移植ではなく、AEGIS-SIGHT のアーキテクチャに合わせて再設計 |
| データ移行 | 移植対象機能に関連するデータのみ移行 |
| 廃棄確認 | 移植しない機能は明示的に廃棄を確認 |
