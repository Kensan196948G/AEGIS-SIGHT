#### Jest→pytest変換マッピング

| Jest | pytest | 備考 |
|------|--------|------|
| `describe()` | `class Test*:` | テストクラスに変換 |
| `it()` / `test()` | `def test_*():` | テスト関数に変換 |
| `beforeEach()` | `@pytest.fixture(autouse=True)` | フィクスチャに変換 |
| `afterEach()` | `yield`フィクスチャ | クリーンアップ |
| `beforeAll()` | `@pytest.fixture(scope="class")` | クラススコープ |
| `expect().toBe()` | `assert x == y` | アサーションに変換 |
| `expect().toThrow()` | `pytest.raises()` | 例外テスト |
| `jest.mock()` | `unittest.mock.patch()` | モック |
| `jest.fn()` | `MagicMock()` | モック関数 |
| `jest.spyOn()` | `mocker.spy()` | スパイ |
| `async/await` | `@pytest.mark.asyncio` | 非同期テスト |

#### 変換難易度別分類（想定）

| 難易度 | 件数（想定） | 説明 |
|--------|-------------|------|
| 自動変換可能 | 約600件 | 単純なアサーション、基本的なモック |
| 半自動（手修正必要） | 約400件 | 複雑なモック、非同期テスト |
| 手動変換必要 | 約157件 | DOM操作、ブラウザ依存、複雑な状態管理 |
