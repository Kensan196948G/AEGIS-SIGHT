| 項目 | 内容 |
|------|------|
| 文書ID | AEGIS-COMP-002 |
| バージョン | 1.0 |
| 作成日 | 2026-03-27 |
| 対象規制 | 金融商品取引法（J-SOX）内部統制報告制度 |

| 対象規模 | 従業員550名、管理端末500台 |

---

## 1. 概要

### 1.1 目的

本文書は、みらい建設工業における J-SOX（金融商品取引法に基づく内部統制報告制度）への対応として、AEGIS-SIGHT が提供する IT 全般統制（ITGC）の設計を定義する。特にライセンス管理、調達・廃棄記録の保存、SAM（Software Asset Management）レポートの自動生成に焦点を当てる。

### 1.2 適用範囲

| 項目 | 内容 |
|------|------|
| 対象業務 | IT資産管理、ソフトウェアライセンス管理、調達・廃棄管理 |
| 対象システム | AEGIS-SIGHT 統合IT資産管理プラットフォーム |
| 対象期間 | 各事業年度（4月〜翌3月） |
| 保存期間 | 調達・廃棄記録7年、監査証跡3年以上 |

### 1.3 関連規格・基準

| 規格 | 関連性 |
|------|--------|
| 金融商品取引法第24条の4の4 | 内部統制報告書提出義務 |
| 企業会計審議会「内部統制基準」 | ITGCの基準 |
| ISO/IEC 27001:2022 | 情報セキュリティ管理策 |
| ISO/IEC 20000-1:2018 | ITサービスマネジメント |
| ISO/IEC 19770-1 | ソフトウェア資産管理 |

## 2. IT全般統制（ITGC）フレームワーク

### 2.1 ITGC 4領域と AEGIS-SIGHT 対応

| ITGC領域 | 統制目的 | AEGIS-SIGHT 対応機能 | 証跡 |
|----------|---------|---------------------|------|
| **プログラム開発** | システム変更の適切な承認・テスト | 変更管理ワークフロー、CI/CDゲート | 変更履歴ログ |
| **プログラム変更** | 本番環境への変更の統制 | 承認フロー、環境分離、デプロイログ | 承認記録・デプロイログ |
| **コンピュータ運用** | 安定したシステム運用 | 監視・アラート、バックアップ自動化 | 運用ログ・障害対応記録 |
| **アクセス管理** | 適切なアクセス権限管理 | RBAC、MFA、アクセスレビュー | アクセスログ・権限変更履歴 |

### 6.2 サンプリング基準

| 母集団サイズ | サンプル数（日次統制） | サンプル数（月次統制） | サンプル数（四半期統制） |

| 1-50 | 全件 | 全件 | 全件 |
| 51-250 | 25件 | 全件 | 全件 |
| 251以上 | 25件 | 2件 | 全件 |

### 6.3 自動統制チェック項目
