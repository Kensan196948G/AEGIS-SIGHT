## 🔧 実装詳細（主要APIパターン）

### ページネーションレスポンス標準

```json
{
  "items": [...],
  "total": 100,
  "offset": 0,
  "limit": 20,
  "has_more": true
}
```

### 認証ヘッダー標準

```
Authorization: Bearer <JWT_TOKEN>
```

### エラーレスポンス標準

| HTTP Status | 意味 | 使用場面 |
|------------|------|---------|
| 401 | 未認証 | 認証ヘッダーなし |
| 403 | 権限不足 | 管理者専用エンドポイント |
| 404 | リソース不在 | 存在しないID指定 |
| 422 | バリデーションエラー | 必須フィールド欠損 |
| 503 | サービス不可 | 外部サービス（M365等）未接続 |
