### E-2: セキュリティ監査（Week 2-5）

| 監査項目 | ツール | 合格基準 |
|---------|--------|---------|
| 依存関係脆弱性スキャン | `safety check`, `npm audit` | Critical/High: 0件 |
| SAST（静的解析） | `bandit`, `semgrep` | Critical: 0件 |
| 認証・認可テスト | 手動テスト + pytest | 全エンドポイント確認 |
| SQLインジェクション対策 | SQLAlchemy ORM確認 | 直接SQL: 0件 |
| XSS対策 | Next.js デフォルト + CSP | CSP設定済み |
| CSRF対策 | JWT + SameSite設定 | 設定確認 |
| 機密情報漏洩チェック | `truffleHog`, `gitleaks` | 0件 |
| APIレート制限 | FastAPI Limiter | 設定確認 |

#### セキュリティ対策チェックリスト

- [ ] 全APIエンドポイントの認証確認（401テスト）
- [ ] 管理者専用エンドポイントの権限確認（403テスト）
- [ ] JWTトークン有効期限設定確認
- [ ] パスワードハッシュ化確認（bcrypt）
- [ ] HTTPS強制設定
- [ ] 環境変数・シークレット管理確認
- [ ] ログに機密情報が出力されていないこと
- [ ] DBアクセスの最小権限確認
- [ ] Docker イメージのセキュリティスキャン
